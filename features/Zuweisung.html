# Assignment

On the main page there is an option Zusweisung in the navigation bar. This takes you to a new page. On this page, you can see on the left side all active Nwks. With regard to Nwks, active means that they are currently still employed as Nwks.
For each Nwk it is possible to see more details by expanding the card using the button provided. In addition, each Nwk has been given a profile picture, which is automatically generated by the initials to make the application more vivid. All Praktikumsstellen from the recently passed Meldezeitraum are displayed on the right-hand side. The Praktikumsstellen are grouped according to their Dienststelle. Each main-Dienststelle can be folded out and then displays a list of all associated Praktikumsstellen. If no Praktikumsstellen from this Dienststelle has been reported, no fold-out element is displayed. If the Dienststelle provided information, that their team has a Planstelle, it will be marked with a special icon (mdi-account-star). Each Praktikumsstellen can be expanded again by clicking on the arrow or the card itself to display more details with all the information about the position.

# Assign Nwks to Praktikumsstellen

To assign a Nwk to a Praktikumsstelle, all that has to be done is drag the Nwk over the desired Praktikumsstelle and drop it there. If everything is all right, then the name of the Nwk will be shown underneath the Praktikumsstelle, the background changes to grey and feedback is given if it was successful. If there is already a Nwk on the desired Praktikumsstelle, then it is not possible to drop another Nwk there.

# Warnings when assigning Nwks

When assigning Nwks, it can happen that the Nwk and Praktikumsstelle don't match in certain criteria. In this case a waring window appear when trying to drop the Nwks onto the Praktikumsstelle.

Warnings that can appear:

  • Nwk is a Studentin but an Auszubildende was requested.
  • Nwk is an Auszubildende but a Studentin was requested.
  • Nwk has a different major than requested
  • Nwk is not the person the Praktikumsstelle specifically requested.
  • Nwk is in a lower study semester than the minimum requested.
  • Nwk is in a lower Lehrjahr than the minimum requested.

⚠ All warnings can be ignored, but it is advised to reevaluate an assignment if a warning appears

# Unassinging Nwks

When a Nwk is assigned to a Praktikumsstelle a little chip with the name of the Nwk shows this assignment. Next to the name you can find a little x. By clicking on this x you can unassign the Nwk. When doing so, a dialog will appear asking you if you really want to continue. When clicking yes the unassigning will be executed and the Nwk will be selectable in the list on the left. When clicking no, the process will be cancelled and nothing will be changed.
