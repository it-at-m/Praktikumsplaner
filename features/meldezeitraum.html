Create a Meldezeitraum ‚Äã

On the page Meldezeitraum it is possible to create a Meldezeitraum.

After clicking, a dialog opens up where the user can enter a name to identify the Meldezeitraum and select a Startzeitpunkt (Startdate) and an Endzeitpunkt (Enddate). These dates are validated so that the Startzeitpunkt cannot be later or the same date as the Endzeitpunkt. Similarly, the Endzeitpunkt cannot be earlier than the Startzeitpunkt.

When clicking on Speichern, the application tries to save the Meldezeitraum and if it was successful, a success message will be displayed. Otherwise, an error message will be displayed.
