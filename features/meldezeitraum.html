Meldezeitraum ​

This page and its actions are only available for örtliche Ausbildungsleitung.

Meldezeitraum Overview ​

The navigation drawer lists the page Meldezeitraum when clicking on this link the user is directed to a page where you can see all Meldezeiträume that have been created so far. They are displayed in three different lists. The first list only shows the current Meldezeitraum. The second list shows all Meldezeiträume that have already ended. The third list shows all Meldezeiträume that have not yet started. If any of the lists is empty, a message will be displayed instead.

Create a Meldezeitraum ​

On the page Meldezeitraum it is possible to create a Meldezeitraum.

After clicking, a dialog opens up where the user can enter a name to identify the Meldezeitraum and select a Startzeitpunkt (Startdate) and an Endzeitpunkt (Enddate). These dates are validated so that the Startzeitpunkt cannot be later or the same date as the Endzeitpunkt. Similarly, the Endzeitpunkt cannot be earlier than the Startzeitpunkt.

When clicking on Speichern, the application tries to save the Meldezeitraum and if it was successful, a success message will be displayed. Otherwise, an error message will be displayed.

Delete a Meldezeitraum ​

On the page Meldezeitraum it is possible to delete a Meldezeitraum.

After clicking on the trash icon, a dialog opens up where the user can confirm the deletion of the Meldezeitraum. Deleting a Meldezeitraum also deletes all Praktikumsplätze that are associated with it, so caution is advised.

When clicking on Ja, the application tries to delete the Meldezeitraum and if it was successful, a success message will be displayed. Otherwise, an error message will be displayed.
