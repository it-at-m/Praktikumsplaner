# Meldezeitraum

# Create Meldezeitraum

On the page [app-url]/#/meldezeitraum it is possible to create a Meldezeitraum.

After clicking a dialog is opening up, where the user can enter a name to identify the Meldezeitraum and select a Startzeitpunkt (Startdate) and an Endzeitpunkt (Enddate). These dates are validated so that the Startzeitpunkt cannot be a later or the same date as the Endzeitpunkt. Similarly, the Endzeitpunkt cannot be earlier than the Startzeitpunkt.

When clicking on Speichern the application tries to save the Meldezeitraum and if it was successful a success message will be displayed. Otherwise, an errormessage will be displayed. The user will be redirected to the landingpage of the application either way.

When clicking the Zurück button the user will be redirected to the landing page.
