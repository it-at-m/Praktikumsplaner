# Assignment

On the main page there is an option `Zusweisung` in the navigation bar.
This takes you to a new page.
On this page, you can see on the left side all active [Nwks](../glossary.mdwk). 
With regard to [Nwks](../glossary.mdwk), active means that they are currently still employed as [Nwks](../glossary.mdwk).  
If a [Nwk](../glossary.mdwk) has details, it is possible to see them by expanding the card using the button provided.
In addition, each [Nwk](../glossary.mdwk) has been given a profile picture, 
which is automatically generated by the initials to make the application more vivid. 
The background of the initials has the [color of the Ausbildungsrichtung](../documentation/architecture/colorpalette/index.md#colors-of-ausbildungsrichtung) or [of the Studiengang](../documentation/architecture/colorpalette/index.md#colors-of-studiengang) of the Nwk.
All [Praktikumsstellen](../glossary.mdraktikumsstelle) from the recently passed [Meldezeitraum](../glossary.md#meldezeitraum) are displayed on the right-hand side. 
The [Praktikumsstellen](../glossary.mdraktikumsstelle) are grouped according to their [Dienststelle](../glossary.md#dienststelle).
Each main-[Dienststelle](../glossary.md#dienststelle) can be folded out and then displays a list of all associated [Praktikumsstellen](../glossary.mdraktikumsstelle). 
If no [Praktikumsstellen](../glossary.mdraktikumsstelle) from this [Dienststelle](../glossary.md#dienststelle) has been reported, no fold-out element is displayed.
If the [Dienststelle](../glossary.md#dienststelle) provided information, that their team has a [Planstelle](../glossary.md#planstelle),
it will be marked with a special icon (mdi-account-star).
Each [Praktikumsstellen](../glossary.mdraktikumsstelle) can be expanded again by clicking on the arrow or the card itself to display more details with all the information about the position.  
Both lists can be scrolled separately.

This page and its actions are only available for [örtliche Ausbildungsleitung](../glossary.md#ortliche-ausbildungsleitung).


## Assign [Nwks](../glossary.mdwk) to [Praktikumsstellen](../glossary.mdraktikumsstelle)

To assign a [Nwk](../glossary.mdwk) to a [Praktikumsstelle](../glossary.mdraktikumsstelle),
all that has to be done is drag the [Nwk](../glossary.mdwk) over the desired [Praktikumsstelle](../glossary.mdraktikumsstelle) and drop it there.
If everything is all right, then the name of the [Nwk](../glossary.mdwk) will be shown underneath the [Praktikumsstelle](../glossary.mdraktikumsstelle),
the background changes to grey and feedback is given if it was successful.
If there is already a [Nwk](../glossary.mdwk) on the desired [Praktikumsstelle](../glossary.mdraktikumsstelle),
then it is not possible to drop another [Nwk](../glossary.mdwk) there.
On the assignment page you can click on a [Nwk](../glossary.mdwk) to see additional information for example the lecture days.
You can also click on a [Praktikumsstelle](../glossary.mdraktikumsstelle) to see additional information about it.
Scrolling works as soon as you press and hold the element at the top or bottom of the screen.

## Warnings when assigning [Nwks](../glossary.mdwk)

When assigning [Nwks](../glossary.mdwk), it can happen that the [Nwk](../glossary.mdwk) and [Praktikumsstelle](../glossary.mdraktikumsstelle) don't match in certain criteria.
In this case a waring window appear when trying to drop the [Nwks](../glossary.mdwk) onto the [Praktikumsstelle](../glossary.mdraktikumsstelle).

Warnings that can appear:
- [Nwk](../glossary.mdwk) is a [Studentin](../glossary.mdtudentin) but an [Auszubildende](../glossary.mduszubildende) was requested.
- [Nwk](../glossary.mdwk) is an [Auszubildende](../glossary.mduszubildende) but a [Studentin](../glossary.mdtudentin) was requested.
- [Nwk](../glossary.mdwk) has a different major than requested
- [Nwk](../glossary.mdwk) is not the person the [Praktikumsstelle](../glossary.mdraktikumsstelle) specifically requested.
- [Nwk](../glossary.mdwk) is not in the requested semester.
- [Nwk](../glossary.mdwk) is not in the requested [Lehrjahr](../glossary.mdehrjahr).

**⚠** All warnings *can* be ignored, but it is advised to reevaluate an assignment if a warning appears

## Unassinging [Nwks](../glossary.mdwk)
When a [Nwk](../glossary.mdwk) is assigned to a [Praktikumsstelle](../glossary.mdraktikumsstelle) a little chip with the name of the Nwk shows this assignment.
This chip has the [color of the Ausbildungsrichtung](../documentation/architecture/colorpalette/index.md#colors-of-ausbildungsrichtung) or [of the Studiengang](../documentation/architecture/colorpalette/index.md#colors-of-studiengang) of the Nwk.
Next to the name you can find a little x. By clicking on this x you can unassign the Nwk. When doing so,
a dialog will appear asking you if you really want to continue. When clicking yes the unassigning will be executed
and the Nwk will be selectable in the list on the left. When clicking no, the process will be cancelled and
nothing will be changed.

## Send successful assignment mails
If the [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung) is satisfied with the assignment of the [Nwks](../glossary.mdwk) to the [Praktikumsstellen](../glossary.mdraktikumsstelle),
then the [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung) can send the assignment mails to the [Örtliche Ausbilderin](../glossary.mdrtliche-ausbilderin).
To do this, the [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung) must click on the `Mails Senden` button.

A Dialog will appear warning about inconsistencies that have been found in the assignments, 
such as a [Praktikumsstelle](../glossary.mdraktikumsstelle) with a "namentliche Anforderung" but without an assigned [Nwk](../glossary.mdwk),
a [Praktikumsstelle](../glossary.mdraktikumsstelle) with priority "DRINGEND" or "ZWINGEND" but without an assigned [Nwk](../glossary.mdwk) or 
a [Nwk](../glossary.mdwk) that is currently not assigned to any [Praktikumsstelle](../glossary.mdraktikumsstelle).
All warnings have to be declared to be okay by clicking `accept` before the user can continue with sending mails.
If just one of the warnings is declined, the whole process will be canceled and the user can correct the assignments. 
After correcting all the warnings that were not solved will show up again and the user has to accept them. 

After the acceptance of all warnings a dialog will appear requesting the [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung) 
to fill in the [Zuweisungszeiträume](../glossary.mduweisungszeitraum). After that they can click on the `Weiter` button.
A dialog will appear asking if the [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung) is sure to send the mails.
Now the mails will be sent to the [Örtliche Ausbilderin](../glossary.mdrtliche-ausbilderin), which got the [Nwks](../glossary.mdwk) assigned to them.
The [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung) will be redirected to the assignment page after that.
If some errors occurred during the sending of the mails, a dialog will appear telling the [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung),
which [Praktikumsstellen](../glossary.mdraktikumsstelle) could not be informed by mail. Currently, there is no way to resend the mails. 
This has to be done manually.
The other mails will still be sent.

## Export assignments as Excel file
The [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung) can export the assignments as an Excel file.
To do this, the [Ausbildungsleitung](../glossary.mdrtliche-ausbildungsleitung) must click on the `Exportieren` button.
After clicking on the button, the file will be downloaded automatically.
Everything about the [Praktikumsstellen](../glossary.mdraktikumsstelle) and the assigned [Nwks](../glossary.mdwk) will be exported.
